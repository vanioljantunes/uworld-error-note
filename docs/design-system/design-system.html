Inter is the primary font family, loaded via Google Fonts CDN with weights 100–900 and font-display: swap. Monospace uses the browser default monospace stack.
IntersectionObserver triggers .reveal and .scale-in classes on scroll. Hero elements use setTimeout cascades on load. All animations respect prefers-reduced-motion.
Dark variants are provided for glass cards (.glass-card-dark), grid dots (.grid-dots-dark), and text stroke (.text-stroke-white). Swap bg-[#EAEAE5] with bg-black and invert text tokens.
