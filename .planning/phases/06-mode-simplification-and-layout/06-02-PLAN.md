  <action>
In `page.module.css`, find `.ankiFormatRow` (around line 3122) and add three properties:

```css
.ankiFormatRow {
  display: flex;
  align-items: center;
  gap: 6px;
  flex-wrap: wrap;           /* LAY-01: wrap to second line on narrow screens */
  row-gap: 4px;              /* tighten vertical spacing for wrapped rows */
  justify-content: flex-end; /* keep wrapped buttons right-aligned */
  margin-left: auto;
}
```

Why `justify-content: flex-end`: The row is right-aligned via `margin-left: auto`. With wrapping, subsequent lines default to `flex-start` without explicit right-alignment.
