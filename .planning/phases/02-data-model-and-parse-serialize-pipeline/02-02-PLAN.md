2. THEN create `gapstrike/src/lib/rebuild-flow-html.ts`:
   - `export function rebuildHTML(graph: FlowGraph): string`
   - Import FlowGraph from `./flowchart-types` and FLOWCHART_STYLES from `./flowchart-styles`
   - Build adjacency: for each edge, map `fromId -> [{toId, stepLabel}]`
   - Build branch lookup: for each branchGroup, map `parentId -> childIds[]`
   - Find root node: the node whose id does not appear as any edge's `toId`
   - Walk from root using a recursive `emitNode(nodeId)` function:
     a. Emit box div: `<div style="${FLOWCHART_STYLES.box}">${node.label}</div>`
     b. Check if nodeId is a branch parent:
        - YES: emit stem, then `<div style="${FLOWCHART_STYLES.branchWrapper}">`, then for each child (left-to-right):
          - Emit corner div (left corner for first child, right corner for last child; if 3 children: left, top-only for middle, right)
          - Emit padding wrapper, then find the edge from parent to this child, emit pill with stepLabel, then stem, then recursively call `emitNode(childId)`
          - Close padding and child outer divs
        - Close branch wrapper div
     c. Check if nodeId has a single outgoing edge (non-branch):
        - Emit stem, pill (with stepLabel from edge), stem, then recursively `emitNode(toId)`
     d. If nodeId has no outgoing edges, stop (leaf node)
   - Wrap everything in outer div: `<div style="${FLOWCHART_STYLES.wrapper}">` + title div + emitNode(root) + `</div>`
   - CRITICAL: All string concatenation uses `""` join — NO newlines between any tags. AnkiDroid converts `\n` to `<br>` on edit, corrupting the structure.
   - For 3-child branch groups: first child gets `branchCornerLeft`, last gets `branchCornerRight`, middle child(ren) get `border-top:2px solid #3a3a3a` only (no left/right border, no margin). Add a comment noting this is a stretch case.
   - The stem div is always wrapped: `<div><div style="${FLOWCHART_STYLES.stem}"></div></div>` (outer div has no style — it's just a block container for the inline stem).

3. Run tests — they MUST PASS (GREEN).

IMPORTANT: Do NOT assert byte-equality between original and rebuilt HTML. Assert semantic equality by comparing the FlowGraph objects from parsing both.
