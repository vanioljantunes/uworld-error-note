**highlightCloze(label: string): React.ReactNode — EXPORT this function as a named export for testing**
- Split on `/({{c\d+::[^}]*}})/g`
- Matched parts get `<span className={styles.clozeHighlight}>{part}</span>`
- Non-matched parts render as plain text
- This ensures FLOW-08: cloze displayed raw with accent highlight

**NodeCard component (inline):**
- Props: `{ node: FlowNode }`
- Renders `<div className={styles.nodeCard}>{highlightCloze(node.label)}</div>`

**EdgePill component (inline):**
- Props: `{ label: string }`
- If label is empty string, return null (no pill for empty labels)
- Renders `<div className={styles.pill}>{label}</div>`

**FlowRenderer component (inline):**
- Props: `{ graph: FlowGraph }`
- Build lookup maps on each render (cheap for small graphs):
  - `toIds = new Set(graph.edges.map(e => e.toId))`
  - `rootNode = graph.nodes.find(n => !toIds.has(n.id))`
  - `nodeById = new Map(graph.nodes.map(n => [n.id, n]))`
  - `branchMap = new Map(graph.branchGroups.map(g => [g.parentId, g.childIds]))`
  - `outgoing = Map<string, FlowEdge[]>` grouped by fromId
- If no rootNode, render `<div className={styles.errorState}>Empty flowchart</div>`
- Recursive `renderNode(nodeId: string)` function with visited Set to prevent cycles:
  1. Emit `<NodeCard node={node} />`
  2. If `branchMap.has(nodeId)`:
     - Emit stem, then `<div className={styles.branchWrapper}>` containing branch arms
     - Each arm: corner div (first child = branchCornerLeft, last = branchCornerRight, middle = branchCornerMiddle), then branchPadding wrapping EdgePill + stem + recursive renderNode(childId)
  3. Else if outgoing edges exist:
     - Emit stem + EdgePill(stepLabel) + stem + renderNode(toId) for first outgoing edge
  4. Leaf nodes: emit nothing after the NodeCard
- Return `<div className={styles.canvas}>{renderNode(rootNode.id)}</div>`
